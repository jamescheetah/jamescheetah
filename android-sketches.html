Cart and Review Checkout

s

The Cart view should have a temporary message describing the replacement policy, as well as has provide the ability for users to indicate to us whether or not they want an item to be replaced.

The checkout view can be accessed by tapping on the "Checkout" button on the Cart view. Several elements are modifiable: The delivery date, order note, selected payment method, delivery note, and promo code state.

Checkout Utils - Delivery Dates

Delivery Date Option A - 1: In this version, we only allow the user to select day but not a time frame. One suggestion is to only show Same-Day and Next Day as they are the most popular, which helps to create a simpler decision. The user can tap on show more days to view additional selectable days.

Delivery Date Option A - 2: After the user taps on the Show More days button, a list of additional days are provided. Note that the additional days don't have a special title as was the case in the Same Day or Next Day options.

Delivery Date Option B - 1:

In this version, we allow the user to select both the date and time frame. Some time frames might not be available.

Delivery Date Option B - 2:

If the user select today, a message should show up noting to the user that there will be an additional surcharge.
